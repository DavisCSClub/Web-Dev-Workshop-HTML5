CSS

Linking Stylesheet


   <link rel="stylesheet" href="style.css">

Then, in the stylesheet....


body {
  background-image: url('...');
}
h1,h2,h3,h4,h5,h6 {
  font-size: 20px;
}

Two ways to custom style elements

By ID's

 <p id="sample_paragraph">This is a sample paragraph</p>
   

 /* Then in the CSS */
 #sample_paragraph {
   color: red;
 }

By Classes


<ul class="squareList">
  <li>Bla1</li> 
  <li>Bla2</li>
</ul>
        

/* Then in the CSS */
.squareList {
  list-style-type: square;
}

  3. Set the p's font-size to 18px
  4. Make the last list of the webpage have square bullets. Use the list-style-type property
